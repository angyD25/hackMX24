# hackMX24

This repository contains an AI-based predictive modeling project focused on forecasting patterns and probabilities of criminal activity in Mexico City using real data.

## Project Overview

- **Goal:** Build, train, and evaluate classification models to predict crime-related patterns.
- **Pipeline:** Extract, transform, and load (ETL) data, then clean and preprocess it for modeling.
- **Output:** Assess model performance and support predictive analysis using real-world crime data.

## Repository Contents

- `THALES_Pipeline.py`: Main pipeline script for data processing and modeling.
- `THALES_Limpieza.py`: Data cleaning and preprocessing routines.
- `Base_de_datos_2024_TS_Predict (1).ipynb`: Notebook with exploratory analysis and model experiments.
- `THALES (1).ipynb`: Additional notebook for analysis and evaluation.

## Usage

1. Review the notebooks for exploratory data analysis and modeling experiments.
2. Run the Python scripts to process data and train models.
3. Inspect the results and performance metrics to validate predictions.

## Notes

- Ensure the required Python packages are installed before running the scripts.
- Verify data paths and file dependencies match your local environment.
